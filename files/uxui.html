PROJECT 1 : REDESIGNING PERSONAL PORTFOLIO

This was a personal project to redesign my portfolio, keeping UX design techniques in mind and really thinking about the process of design. Optimizing the site across multiple platforms was one of the main goals of this project.

I wanted to rethink the design of my previous site. It was a bit overwhelming, showcasing everything visual I've accumulated, including all artwork and photos. I wanted to impress the viewer with animations at every scroll and presenting everything on one page so that everyone would be able to see everything in one go. It was lacking directness in presentation.

  1. My first step was identify the changes I wanted to make:
    • remove elements not entirely relevant to purpose of site
      • introduce heirarchy and ability to view certain parts of page
    • prioritize experience on mobile devices
    • eliminate elements that are not supported by all browsers

  2. I then made sketches to facilitate the developing process. Having a clear visual goal sped up the front-end development.

  3. I developed the site using html/css + javascript. Code available on my github.
